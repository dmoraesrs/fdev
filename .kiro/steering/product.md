# Product Vision - FDEV Toolkit

## Propósito

FDEV é um **toolkit de desenvolvimento com AI** — um conjunto de workflows, configs, templates e guardrails que os times adotam para trabalhar com Kiro IDE e Claude Code de forma padronizada.

## Público-Alvo

- Times de desenvolvimento da empresa
- Tech Leads e arquitetos
- Desenvolvedores adotando AI-assisted development

## O que o FDEV fornece

| Categoria | O que é | Para que serve |
|-----------|---------|----------------|
| **Workflows** | MDs com padrões para cada fase do dev | Guiar o AI (Kiro/Claude) em cada etapa |
| **Skills** | Slash commands (/planejar, /codificar, etc.) | Invocar workflows rapidamente |
| **Templates** | Configs prontas (.pre-commit, .editorconfig, specs) | Setup rápido de projetos |
| **MCPs** | Context7 + Serena configurados | Docs on-demand + navegação semântica |
| **Guardrails** | Pre-commit, linters, CI | Qualidade e segurança automáticas |
| **Kiro configs** | Steering, hooks, MCP settings | Contexto persistente no IDE |

## Analogia

- **Repo de Agentes** = toolkit para **infra/operações** (agentes: k8s, devops, secops...)
- **FDEV** = toolkit para **desenvolvimento** (workflows: planejar, codificar, revisar...)
